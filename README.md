# QR Code Generator
A simple and practical project for generating QR Codes using user-entered text or links.

## Features
- Instant QR Code generation by entering text or URL
- Beautiful and simple user interface
- Responsive design
- Uses the free qrserver API
- No libraries or frameworks required

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)

## How it Works
- Enter the text or link in the input field.
- Click the “Generate QR Code” button.
- The QR Code will be automatically generated and displayed.
- The QR section will hide again when the input is cleared.
