# 📱 QR Code Generator

A clean, responsive web application that instantly generates QR codes from any text or URL. Built with vanilla HTML, CSS, and JavaScript.
### 🚀 Live Demo

[![Live Demo](https://img.shields.io/badge/Live%20Demo-View%20Project-3498DB?style=for-the-badge&logo=github&logoColor=white)](https://shahzadhpr.github.io/QR-Code-Generator)

> 👆🏻 Click here to see the live version of the project

&nbsp;

## ✨ Features

- 🔗 Generate QR codes from any URL or text input
- ⚡ Real-time generation with instant preview
- 📱 Fully responsive design – works on desktop, tablet, and mobile
- 🎨 Clean, modern UI with smooth animations
- ♿ Accessible and user-friendly interface

&nbsp;

## 🛠️ Technologies Used

- **HTML5** – Semantic structure
- **CSS3** – Modern styling with Flexbox, Google Fonts, and responsive design
- **JavaScript (Vanilla)** – DOM manipulation and event handling
- **QR Server API** – [goqr.me API](https://goqr.me/api/) for QR code generation

&nbsp;

## 📦 Installation
```bash
git clone https://github.com/yourusername/Qr-Code-Generator.git

cd Qr-Code-Generator
```
Then open the index.html file in your browser.

&nbsp;

## 💻 Usage
- Enter any text or URL in the input field

- Click the "Generate QR Code" button

- Your QR code will appear instantly

- Scan the QR code with any QR reader to verify the content
